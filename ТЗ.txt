ТЕХНИЧЕСКОЕ ЗАДАНИЕ
Проект: FinTrc — Трекер личных финансов
Тип: Веб-приложение (Python/Flask)

1. НАЗНАЧЕНИЕ
Автоматизация учёта личных доходов и расходов с возможностью анализа баланса, фильтрации операций, конвертации валют и работы с данными через CSV.

2. ОСНОВНОЙ ФУНКЦИОНАЛ
- Аутентификация: регистрация, вход/выход, сессии, хеширование паролей.
- Транзакции: добавление, просмотр истории, редактирование, удаление по ID.
- Аналитика: дашборд (баланс, итоги доходов/расходов, 5 последних операций).
- Фильтрация: переключение между "Все", "Доходы", "Расходы".
- Валюты: онлайн-конвертер через внешний API (exchangerate-api).
- Данные: экспорт таблицы в CSV, импорт транзакций из CSV-файла с валидацией.

3. ТЕХНОЛОГИЧЕСКИЙ СТЕК
- Backend: Python 3.9+, Flask, SQLAlchemy (чистый ORM без Flask-SQLAlchemy)
- БД: SQLite (локальный файл, авто-создание таблиц)
- Frontend: HTML5, Jinja2-шаблоны, базовая вёрстка (extends base.html)
- Сторонние библиотеки: requests (API), werkzeug (хеширование паролей)

4. УСТАНОВКА И ЗАПУСК
1. pip install -r requirements.txt
2. python app.py (сервер запустится на http://127.0.0.1:5000)
3. БД fintrc.db создаётся автоматически при первом запуске.

5. ОГРАНИЧЕНИЯ
- Конвертер требует активного интернет-соединения.
